# ATP-PCA

Projeto em Python para análise de componentes principais (PCA) aplicado a dados de ATP (adenosina trifosfato) ou séries temporais/biológicas similares. Este repositório oferece scripts e utilitários para pré-processamento, execução de PCA, visualização e interpretação dos componentes principais.

## Funcionalidades
 - Carregamento e limpeza de dados (CSV, Excel ou arrays numpy)
 - Normalização e tratamento de valores ausentes
 - Cálculo de PCA com scikit-learn
 - Visualizações: scree plot, biplot, projeção 2D/3D
 - Exportação de componentes e reconstruções

## Requisitos
 - Python 3
 - numpy
 - pandas
 - scikit-learn
 - matplotlib
 - seaborn
